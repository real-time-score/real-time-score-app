## 작업 방식

### 문서 작성 프로세스
1. **PDF 페이지 확인**: 해당 섹션의 원본 PDF 페이지 참조
2. **화면 분석**: UI 구성요소, 레이아웃 파악
3. **기능 정의**: 사용자 시나리오, 인터랙션 정리
4. **데이터 도출**: 필요한 데이터 필드 정의
5. **API 설계**: Admin에서 필요한 API 엔드포인트 정리
6. **검토 및 보완**: 누락된 항목 체크

### 문서 업데이트 규칙
- 문서 작성 완료 시 체크리스트 업데이트 `[ ]` → `[x]`
- 진행률 갱신
- 작업 로그에 기록 추가
- 최종 수정일 업데이트

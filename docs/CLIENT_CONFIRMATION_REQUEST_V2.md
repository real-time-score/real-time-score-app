## 2. 전문가/픽 상세 기획

### 2.1 전문가 등급 체계

> ⚠️ **1차 응답**: 조건 설정 필요

| 항목 | 질문 | 결정 |
|------|------|------|
| 등급 종류 | 몇 가지 등급으로 구분? (예: 골드/실버/브론즈 3단계) | |
| 등급 산정 기준 | 적중률? 활동량? 판매 수? 복합 기준? | |
| 등급 갱신 주기 | 주간? 월간? 분기? | |

### 2.2 픽 가격 정책

> ⚠️ **1차 응답**: 등급별 세부 설정 필요

| 항목 | 질문 | 결정 |
|------|------|------|
| 전체 가격 범위 | 유료픽 최소/최대 가격 (예: 100C ~ 10,000C) | |
| 등급별 가격 제한 | 등급에 따라 설정 가능한 가격 범위 차등 적용? | |
| 무료픽 일일 한도 | 전문가당 하루 무료픽 등록 제한? (예: 3개/일) | |

### 2.3 픽 전문가 조건

> ⚠️ **1차 응답**: 픽 등록 조건 설정 필요, 픽 전문가 자격 조건 설정 필요

| 항목 | 질문 | 결정 |
|------|------|------|
| 등록 자격 | 픽 전문가가 되기 위한 조건 (예: 가입 후 30일, 예측 10회 이상) | |
| 신규 전문가 제한 | 신규 전문가의 초기 활동 제한 (예: 무료픽만 10회 등록 후 유료 가능) | |

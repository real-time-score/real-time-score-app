# 약관

## 1. 개요

서비스 이용약관, 개인정보처리방침 등 법적 약관을 조회할 수 있는 화면입니다.

**원본 PDF 참조**: 81-83페이지

---

## 9. 데이터 요구사항

### 9.1 약관 데이터

```typescript
interface Terms {
  id: string;
  type: TermsType;
  title: string;
  content: string;             // HTML 또는 Markdown
  version: string;             // e.g., "v2.0"
  effectiveDate: string;       // 시행일
  isActive: boolean;           // 현재 적용 여부
  createdAt: string;
  updatedAt: string;
}

type TermsType =
  | 'service'                  // 서비스 이용약관
  | 'privacy'                  // 개인정보 처리방침
  | 'location'                 // 위치기반서비스 약관
  | 'youth'                    // 청소년 보호정책
  | 'opensource';              // 오픈소스 라이선스
```

### 9.2 약관 동의 이력

```typescript
interface TermsAgreement {
  id: string;
  userId: string;
  termsId: string;
  termsType: TermsType;
  termsVersion: string;
  agreedAt: string;
}
```

### 9.3 오픈소스 라이선스

```typescript
interface OpenSourceLicense {
  id: string;

  copyright: string;           // 저작권 표시
```

## 12. 비즈니스 로직

### 12.1 약관 변경 고지

```
1. 새 약관 버전 등록 (시행일 설정)
2. 시행 7일 전 사용자 알림 발송
3. 시행일에 자동 활성화
4. 로그인 시 재동의 팝업 표시 (필수 약관)
5. 동의 내역 기록
```

### 12.2 필수 약관 미동의 처리

```
1. 로그인 시 필수 약관 동의 여부 확인
2. 미동의 약관 있을 경우 약관 동의 화면으로 이동
3. 동의 완료 시 서비스 이용 가능
4. 미동의 시 서비스 이용 제한
```

### 12.3 약관 동의 철회

```
- 선택 약관: 설정에서 철회 가능
- 필수 약관: 회원탈퇴로만 철회 가능
```

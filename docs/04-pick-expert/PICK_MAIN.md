# 픽전문가 메인 화면

## 1. 개요

픽전문가 메인 화면은 유료픽과 무료픽을 탐색하고, 전문가들의 분석 픽을 확인할 수 있는 화면입니다.

- **원본 PDF 참조**: 45-46페이지

---

## 2. 화면 구성

### 2.1 전체 레이아웃

```
┌─────────────────────────────────────┐
│    픽 전문가              MY 픽    │  ← 헤더
├─────────────────────────────────────┤
│   [유료픽]      무료픽              │  ← 탭 (활성탭: 네이비 배경)
├─────────────────────────────────────┤
│ 전체  e스포츠  축구  야구  농구 ... │  ← 종목 카테고리
├─────────────────────────────────────┤
│ 전체  카테고리1  카테고리2  ...     │  ← 서브 카테고리
├─────────────────────────────────────┤
│ ┌───────────────────────────────┐   │
│ │ 🔍 전문가명, 경기명을 입력하세요│  │  ← 검색창
│ └───────────────────────────────┘   │
├─────────────────────────────────────┤
│ ┌───────────────────────────────┐   │
│ │ [프로필]  전문가명        🔔  │   │
│ │   최근 N경기중 N경기 NNN적중   │   │
│ │   구매 NNN  N.NNN             │   │  ← 전문가 카드
│ │ ───────────────────────────   │   │
│ │   [팀A 로고]  7/11(금)        │   │
│ │               15:30           │   │
│ │   팀명        팀명            │   │
│ │ ───────────────────────────   │   │
│ │   타이틀 노출                 │   │
│ │ ───────────────────────────   │   │
│ │   승 N배    +N.N%    N.N%    │   │
│ │   승패  1  핸디  U/O    (+)  │   │
│ └───────────────────────────────┘   │
│                                     │
├─────────────────────────────────────┤
│  홈  픽전문가  커뮤니티  랭킹  MY   │  ← 하단 탭바
└─────────────────────────────────────┘
```

> **디자인 참고**: 탭바 "픽전문가"가 선택된 상태, (+) 버튼은 픽 등록 플로팅 버튼

### 2.2 상단 영역
| 요소 | 설명 |
|------|------|
| 헤더 | "픽 전문가" 타이틀 (좌측) + "MY 픽" 버튼 (우측) |
| 탭 바 | 유료픽 / 무료픽 전환 탭 (활성탭: 네이비 배경) |

### 2.3 필터 영역
| 요소 | 설명 |
|------|------|
| 종목 카테고리 | 전체, e스포츠, 축구, 야구, 농구 등 |
| 서브 카테고리 | 선택된 종목의 하위 카테고리 |
| 검색창 | "전문가명, 경기명을 입력하세요" 플레이스홀더 |

### 2.4 전문가 카드
각 픽 카드에 표시되는 정보:

| 요소 | 설명 |
|------|------|
| 프로필 이미지 | 전문가 프로필 사진 |
| 전문가명 | 전문가 닉네임 |
| 알림 아이콘 | 🔔 - 전문가 알림 설정 |
| 적중률 정보 | "최근 N경기중 N경기 NNN적중" |
| 구매/포인트 | 구매수, 필요 포인트 |
| 경기 정보 | 팀 로고, 경기일시, 팀명 |
| 타이틀 | 픽 타이틀/설명 |
| 배당 정보 | 승패, 핸디, U/O 선택값 및 배당률 |

### 2.5 플로팅 버튼
| 요소 | 설명 |
|------|------|
| (+) 버튼 | 픽 등록 화면으로 이동 (전문가 조건 충족 시)

## 3. 기능 상세

### 3.1 탭 전환
- **유료픽 탭**: 포인트를 사용해 구매하는 프리미엄 픽 목록
- **무료픽 탭**: 무료로 열람 가능한 픽 목록
- 탭 전환 시 해당 카테고리의 픽 목록 로드

### 3.2 종목 필터
- 가로 스크롤 가능한 칩 형태의 필터
- 선택된 종목에 해당하는 픽만 필터링
- 기본값: "전체"

### 3.3 검색 기능
- 검색 아이콘 클릭 시 검색 UI 표시
- 검색 대상: 픽 제목, 전문가 닉네임, 팀명

### 3.4 픽 카드 인터랙션
- 카드 클릭 시 픽 상세 화면으로 이동
- 유료픽: 미구매 시 포인트 결제 팝업 표시
- 무료픽: 바로 상세 내용 확인 가능

### 3.5 MY픽 버튼
- 플로팅 버튼으로 항상 화면 하단에 표시
- 클릭 시 MY픽 화면으로 이동

### 3.6 유료픽 → 무료픽 전환
- 경기 종료 후 유료픽은 자동으로 무료픽으로 전환됨
- 전환된 픽은 무료픽 탭에서도 조회 가능

## 4. 데이터 요구사항

### 4.1 픽 목록 데이터
```typescript
interface Pick {
  id: string;
  type: 'paid' | 'free';
  expert: {
    id: string;
    nickname: string;
    profileImage: string;
    grade: string;  // 등급 (예: 골드, 실버 등)
  };
  title: string;
  match: {
    id: string;
    homeTeam: string;
    awayTeam: string;
    league: string;
    matchTime: string;
    status: 'scheduled' | 'live' | 'finished';
  };
  sport: 'soccer' | 'baseball' | 'basketball' | 'volleyball' | 'hockey';
  bettingType: 'match_result' | 'handicap' | 'under_over';
  prediction: string;  // 예측 값 (예: "홈팀 승", "오버 2.5" 등)
  points: number;  // 유료픽 필요 포인트 (무료픽은 0)
  result?: 'hit' | 'miss' | null;  // 적중 여부
  viewCount: number;
  likeCount: number;
  createdAt: string;
}
```

interface SportFilter {

## 5. API 요구사항

### 5.1 픽 목록 조회
```
GET /api/picks
```

**Query Parameters:**
| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| type | string | O | 'paid' 또는 'free' |
| sport | string | X | 스포츠 종목 코드 |
| keyword | string | X | 검색 키워드 |
| page | number | X | 페이지 번호 (기본값: 1) |
| limit | number | X | 페이지당 개수 (기본값: 20) |

**Response:**
```json
{
  "success": true,
  "data": {
    "picks": [Pick],
    "pagination": {
      "currentPage": 1,
      "totalPages": 10,
      "totalCount": 200
    }
  }
}
```

### 5.2 종목 필터 목록 조회
```
GET /api/picks/sports
```

**Response:**
```json
{
  "success": true,
  "data": [
    { "code": "all", "name": "전체", "icon": "all_icon" },
    { "code": "soccer", "name": "축구", "icon": "soccer_icon" },
    { "code": "baseball", "name": "야구", "icon": "baseball_icon" }
  ]
}
```

## 7. 화면 상태

### 7.1 로딩 상태
- 스켈레톤 UI로 픽 카드 형태 표시

### 7.2 빈 상태
- 해당 조건의 픽이 없을 경우 빈 상태 메시지 표시
- "등록된 픽이 없습니다"

### 7.3 에러 상태
- 네트워크 오류 시 재시도 버튼과 함께 에러 메시지 표시

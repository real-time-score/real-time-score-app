# 공통 컴포넌트 (Common Components)

## 1. 개요

이 문서는 스포츠스코어픽 앱에서 사용되는 공통 UI 컴포넌트를 정의합니다.

- **원본 PDF 참조**: 6페이지 (공통 모달)
- **적용 범위**: 앱 전역

---

## 2. 공통 모달 (Common Modal)

### 2.1 모달 타입

| 타입 | 설명 | 버튼 구성 |
|------|------|----------|
| Confirm Modal | 사용자 확인이 필요한 액션 | 취소, 확인 (2버튼) |
| Alert Modal | 단순 안내/알림 | 확인 (1버튼) |

### 2.2 모달 구성 요소

```
┌─────────────────────────────────┐
│                                 │
│           [타이틀]              │
│                                 │
│         [설명 텍스트]           │
│      (멀티라인 가능)            │
│                                 │
│   ┌─────────┐  ┌─────────┐     │
│   │  취소   │  │  확인   │     │
│   └─────────┘  └─────────┘     │
└─────────────────────────────────┘
```

### 2.3 모달 스타일

| 속성 | 값 |
|------|-----|
| 배경 | Dimmed (반투명 검정) |
| 모달 배경 | 흰색 |
| 모서리 | Rounded (라운드 처리) |
| 버튼 배치 | 하단 정렬, 좌우 나란히 |

### 2.4 버튼 스타일

| 버튼 | 스타일 | 색상 |
|------|--------|------|
| 취소 | Secondary (보조) | 회색 계열 |
| 확인 | Primary (주요) | 브랜드 컬러 |

### 2.5 인터랙션

- **배경 탭**: Dimmed 영역 탭 시 모달 닫힘 (선택적)
- **취소 버튼**: 모달 닫힘, 액션 취소
- **확인 버튼**: 해당 액션 수행 후 모달 닫힘

## 3. 사용 케이스

### 3.1 Confirm Modal 사용 예시

| 상황 | 타이틀 | 설명 |
|------|--------|------|
| 로그아웃 | 로그아웃 | 로그아웃 하시겠습니까? |
| 게시글 삭제 | 삭제 | 게시글을 삭제하시겠습니까? |
| 픽 구매 | 구매 확인 | 해당 픽을 구매하시겠습니까? |
| 신고 | 신고하기 | 해당 게시글을 신고하시겠습니까? |

### 3.2 Alert Modal 사용 예시

| 상황 | 타이틀 | 설명 |
|------|--------|------|
| 포인트 부족 | 포인트 부족 | 포인트가 부족합니다. |
| 네트워크 오류 | 오류 | 네트워크 연결을 확인해주세요. |
| 완료 안내 | 완료 | 정상적으로 처리되었습니다. |

## 4. 데이터 요구사항

### 4.1 Modal 데이터 구조

```typescript
interface ModalData {
  type: 'confirm' | 'alert';
  title: string;
  description: string;
  confirmText?: string;      // 기본값: "확인"
  cancelText?: string;       // 기본값: "취소"
  onConfirm: () => void;
  onCancel?: () => void;
  dismissOnBackgroundTap?: boolean;  // 기본값: true
}
```

---

## 5. API 요구사항

공통 모달은 프론트엔드 전용 컴포넌트로, 별도의 API 요구사항 없음.

---

## 6. 관리자 기능

공통 모달 문구의 경우 하드코딩되며, 관리자 페이지에서 별도 관리 불필요.
단, 향후 다국어 지원 시 번역 관리 필요.

# 로딩/스플래시 화면 (Loading/Splash)

## 1. 개요

이 문서는 앱 시작 시 표시되는 스플래시 화면과 로딩 상태를 정의합니다.

- **원본 PDF 참조**: 7-8페이지 (스플래시 화면)
- **적용 범위**: 앱 시작, 로딩 상태

---

## 2. 스플래시 화면 (Splash Screen)

### 2.1 화면 구성

```
┌──────────────────────────────────────┐
│                                      │
│                                      │
│                                      │
│            ┌─────────┐               │
│            │  REAL   │               │
│            │  TIME   │               │
│            │  score  │               │
│            └─────────┘               │
│                                      │
│        リアルタイムスコア            │
│        (REAL TIME SCORE)             │
│                                      │
│                                      │
│                                      │
└──────────────────────────────────────┘
```

### 2.2 구성 요소

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 배경 | 전체 화면 배경 | 흰색 (#FFFFFF) |
| 앱 로고 | "REAL TIME score" 로고 | 중앙 정렬, 파란색 박스 형태 |
| 서비스명 | "リアルタイムスコア" (일본어) | 중앙 정렬, 네이비 컬러, 아이콘 하단 |

> **디자인 참고**: 로고는 3줄 구조 ("REAL" / "TIME" / "score")의 파란색 박스 형태

### 2.3 표시 시간

| 상황 | 최소 표시 시간 | 최대 표시 시간 |
|------|--------------|--------------|
| 앱 초기화 | 1.5초 | 3초 |
| 데이터 로딩 완료 시 | 즉시 종료 가능 | - |
| 네트워크 지연 시 | - | 10초 (타임아웃) |

### 2.4 동작 플로우

```
앱 실행
    ↓
스플래시 표시
    ↓
초기화 작업 수행
├── 앱 설정 로드
├── 사용자 인증 상태 확인
├── 필수 데이터 프리로드
└── 푸시 토큰 등록
    ↓
초기화 완료
    ↓
메인 화면 또는 온보딩 화면으로 전환
```

## 3. 로딩 인디케이터

### 3.1 로딩 타입

| 타입 | 사용 케이스 | UI 형태 |
|------|------------|---------|
| 전체 화면 로딩 | 화면 전환, 초기 로드 | 중앙 스피너 + 딤드 배경 |
| 인라인 로딩 | 리스트 로딩, 더보기 | 컨텐츠 영역 내 스피너 |
| 스켈레톤 로딩 | 컨텐츠 로드 대기 | 플레이스홀더 UI |
| 버튼 로딩 | 액션 처리 중 | 버튼 내 스피너 |

## 4. 에러 상태 처리

### 4.1 네트워크 에러

| 상태 | 표시 내용 | 액션 |
|------|----------|------|
| 연결 실패 | "네트워크 연결을 확인해주세요" | 재시도 버튼 |
| 타임아웃 | "응답 시간이 초과되었습니다" | 재시도 버튼 |
| 서버 오류 | "일시적인 오류가 발생했습니다" | 재시도 버튼 |

## 5. 데이터 요구사항

### 5.1 스플래시 데이터

```typescript
interface SplashConfig {
  minDisplayTime: number;    // 최소 표시 시간 (ms)
  maxDisplayTime: number;    // 최대 표시 시간 (ms)
  logoUrl?: string;          // 동적 로고 URL (선택)
  backgroundColor?: string;  // 배경색 (선택)
}
```

### 5.2 로딩 상태

```typescript
interface LoadingState {
  isLoading: boolean;
  type: 'fullscreen' | 'inline' | 'skeleton' | 'button';
  message?: string;
}
```

## 6. API 요구사항

### 6.1 앱 초기화 API

| 엔드포인트 | 메서드 | 설명 |
|-----------|--------|------|
| `/api/app/init` | GET | 앱 초기화 데이터 조회 |

#### Response

```json
{
  "success": true,
  "data": {
    "minAppVersion": "1.0.0",
    "forceUpdate": false,
    "maintenance": false,
    "maintenanceMessage": null,
    "config": {
      "splashMinTime": 1500
    }
  }
}
```

## 7. 관리자 기능

| 항목 | 관리 필요 | 설명 |
|------|----------|------|
| 스플래시 이미지 | △ | 시즌별 변경 가능 |

|-----------|--------|------|

| `/admin/api/app/config` | PUT | 앱 설정 수정 |
| `/admin/api/app/maintenance` | POST | 점검 모드 설정 |

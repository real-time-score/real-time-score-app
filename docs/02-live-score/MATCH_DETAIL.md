# 경기 상세 (MATCH_DETAIL)

## 1. 개요

경기 상세 화면은 특정 경기의 모든 정보를 확인할 수 있는 페이지입니다. 실시간 스코어, 경기 통계, 라인업, H2H, 배당 정보, AI 예측, 전문가 픽 등 다양한 정보를 탭 형태로 제공합니다.

**원본 PDF 참조**: 20-36페이지

---

## 2. 화면 구성

### 2.1 전체 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  [로고]                              ♡  🔔  ⋮      │  ← 헤더
├─────────────────────────────────────────────────────┤
│   [경기]  [배당정보]  [채팅방]  [적정금]            │  ← 메인 탭
├─────────────────────────────────────────────────────┤
│                                                     │
│  운영자 등록글 노출                                 │  ← 운영자 코멘트
│                                                     │
│  [홈팀 로고]    1  vs  5    [원정팀 로고]          │
│   홈팀명                       원정팀명             │  ← 스코어보드
│                                                     │
│  LIVE  N.NN  N.NN  N.NN                           │  ← 배당 요약
│                                                     │
├─────────────────────────────────────────────────────┤
│                                                     │
│              탭 콘텐츠 영역                         │
│              (스크롤 영역)                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

> **디자인 참고**: 경기 상세 화면에서 메인 탭은 "경기/배당정보/채팅방/적정금" 4개로 구성

### 2.2 헤더 구성

| 요소 | 설명 |
|-----|------|
| 로고 | 앱 로고 (좌측) |
| 즐겨찾기 | ♡ 아이콘 - 경기 즐겨찾기 추가 |
| 알림 | 🔔 아이콘 - 경기 알림 설정 |
| 더보기 | ⋮ 아이콘 - 공유, 신고 등 추가 기능 |

### 2.3 스코어보드

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  [🔔]  [⭐]                                         │  ← 알림/즐겨찾기

│   전북 현대               울산 현대                 │

│              2  -  1                                │

│           후반 67' (진행중)                         │  ← 경기 상태
│                                                     │
│  ┌─────────────────────────────────────────┐       │
│  │ 국내배당  1.85  3.20  4.50              │       │  ← 배당 정보
│  │ 해외배당  1.82  3.25  4.60              │       │
│  └─────────────────────────────────────────┘       │

```

## 3. 탭 메뉴 상세

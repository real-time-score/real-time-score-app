# 선수 상세 (PLAYER_DETAIL)

## 1. 개요

선수 상세 화면은 특정 선수의 전체 정보를 확인할 수 있는 페이지입니다. 선수 기본 정보, 소속팀, 시즌 통계, 경기 기록, 커리어 정보 등 선수에 관한 모든 정보를 제공합니다.

**원본 PDF 참조**: 41-42페이지

## 6. 데이터 요구사항

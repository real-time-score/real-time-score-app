# 팀 상세 (TEAM_DETAIL)

## 1. 개요

팀 상세 화면은 특정 팀의 전체 정보를 확인할 수 있는 페이지입니다. 팀 기본 정보, 선수단 명단, 최근 경기 결과, 시즌 통계, 예정 경기 등 팀에 관한 모든 정보를 제공합니다.

**원본 PDF 참조**: 37-40페이지

```
GET /api/v1/teams/{team_id}/squad
```

**Query Parameters**:

| 파라미터 | 타입 | 필수 | 설명 |
|---------|------|------|------|
| season | String | N | 시즌 (default: 현재 시즌) |
| position | String | N | 포지션 필터 |

**Response**:

```json
{
  "success": true,
  "data": {
    "team_id": "TEAM001",
    "season": "2025",
    "squad": [
      {
        "player_id": "PLAYER001",

      "GK": 3,
```

### 7.3 팀 경기 일정 조회

```
GET /api/v1/teams/{team_id}/matches
```

**Query Parameters**:

| 파라미터 | 타입 | 필수 | 설명 |
|---------|------|------|------|
| type | String | N | upcoming/past/all (default: all) |
| season | String | N | 시즌 |
| limit | Integer | N | 조회 개수 (default: 10) |

### 7.4 팀 통계 조회

```
GET /api/v1/teams/{team_id}/statistics
```

**Query Parameters**:

| 파라미터 | 타입 | 필수 | 설명 |
|---------|------|------|------|
| season | String | N | 시즌 (default: 현재 시즌) |
| league_id | String | N | 리그 ID |

## 8. 관리자 기능

### 8.1 팀 정보 관리

| 기능 | 설명 |
|-----|------|
| 팀 정보 등록 | 신규 팀 정보 등록 |
| 팀 정보 수정 | 팀 기본 정보 수정 |
| 로고 업로드 | 팀 엠블럼 이미지 업로드 |
| 선수단 관리 | 소속 선수 추가/삭제 |

### 8.2 Admin API

| 메서드 | 엔드포인트 | 설명 |
|-------|-----------|------|
| POST | /admin/api/v1/teams | 팀 등록 |
| PUT | /admin/api/v1/teams/{id} | 팀 정보 수정 |
| DELETE | /admin/api/v1/teams/{id} | 팀 삭제 |
| POST | /admin/api/v1/teams/{id}/logo | 로고 업로드 |
| POST | /admin/api/v1/teams/{id}/players | 선수 추가 |
| DELETE | /admin/api/v1/teams/{id}/players/{player_id} | 선수 삭제 |

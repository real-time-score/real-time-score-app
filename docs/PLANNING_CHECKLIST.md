# 기획 확인 필요 사항 체크리스트

> 최종 업데이트: 2025-12-23 (admin docs 반영)
>
> 이 문서는 모바일 앱 스펙 문서 작성 과정에서 발견된 기획적 확인 필요 사항을 정리한 것입니다.
> 개발 착수 전 기획자/PM과 협의하여 확정해야 합니다.
>
> **참고**: Admin 스펙 문서 (`/admin/docs/`)에서 확정된 사항은 `[x]`로 표시하고 출처를 명시했습니다.

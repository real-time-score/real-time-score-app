# 컨텐츠 관리 (Content Management)

## 1. 개요

이 문서는 스포츠스코어픽 Admin에서 관리하는 모든 컨텐츠(배너, 공지사항, 이벤트, FAQ, 약관, 게시글, 댓글, 픽)를 정의합니다.

- **원본 PDF 참조**: 13페이지 (배너), 53-57페이지 (커뮤니티), 74-83페이지 (공지/FAQ/약관)
- **적용 범위**: Admin > 컨텐츠 관리, 커뮤니티 관리 메뉴

## 2. 배너 관리

### 2.1 배너 유형

PDF 13페이지 참조:

| 유형 | 위치 | 크기 비율 |
|------|------|-----------|
| 메인 배너 | 라이브 메인 상단 | 16:9 또는 2:1 |
| 이벤트 배너 | 이벤트 페이지 | 16:9 |
| 팝업 배너 | 앱 진입 시 | 1:1 또는 4:5 |

### 2.2 배너 목록 화면

```
┌─────────────────────────────────────────────────────────────┐
│ 배너 관리                                    [+ 배너 등록]   │
├─────────────────────────────────────────────────────────────┤

├─────────────────────────────────────────────────────────────┤
│ □ │ ID │ 썸네일 │ 제목 │ 유형 │ 상태 │ 노출기간 │ 순서 │ 관리│
├───┼────┼────────┼──────┼──────┼──────┼──────────┼──────┼────┤
│ □ │ 1  │ [img]  │ 신규 │ 메인 │ 노출 │ 12/1~31  │ 1    │ [▲▼]│
│ □ │ 2  │ [img]  │ 이벤 │ 이벤트│ 대기 │ 12/15~31 │ 2    │ [▲▼]│
│ ...                                                         │
├─────────────────────────────────────────────────────────────┤
│ [선택 삭제] [선택 숨김]                                      │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 배너 등록/수정 화면

| 필드 | 타입 | 필수 | 설명 |
|------|------|:----:|------|
| 배너 유형 | Select | ✅ | 메인/이벤트/팝업 |
| 제목 | Text | ✅ | 관리용 제목 |
| 이미지 | Image | ✅ | 배너 이미지 |
| 링크 유형 | Select | ✅ | 외부링크/앱내링크/없음 |
| 링크 URL | Text | - | 링크 유형에 따라 |
| 노출 상태 | Toggle | ✅ | 노출/숨김 |
| 노출 시작일 | DateTime | ✅ | 노출 시작 일시 |
| 노출 종료일 | DateTime | ✅ | 노출 종료 일시 |
| 노출 순서 | Number | ✅ | 정렬 순서 |

## 9. 신고 관리

**✅ 확정 정책 (2026-01-06)**:
- 실시간 채팅 신고 **5회 누적** → 해당 채팅 **삭제** 처리

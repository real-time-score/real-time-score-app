# 통계/대시보드 (Statistics & Dashboard)

## 1. 개요

이 문서는 스포츠스코어픽 Admin의 통계 및 대시보드 기능을 정의합니다.

- **원본 PDF 참조**: 앱 전반의 데이터를 기반으로 관리자용 통계 도출
- **적용 범위**: Admin > 대시보드, 통계 메뉴

---

## 2. 대시보드

### 2.1 대시보드 구성

관리자 로그인 시 첫 화면으로 표시되는 주요 지표 요약

```
┌─────────────────────────────────────────────────────────────────────┐
│ 대시보드                                        2025-12-22 (월)     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐│
│  │  오늘 가입    │ │  오늘 DAU    │ │  오늘 게시글  │ │  오늘 매출   ││
│  │     125      │ │    8,432     │ │     342      │ │   ₩1,250,000 ││
│  │   +12.5%↑   │ │   -2.3%↓    │ │   +8.1%↑    │ │   +15.2%↑   ││
│  └──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘│
│                                                                     │
│  ┌────────────────────────────────┐ ┌────────────────────────────┐  │
│  │       주간 가입자 추이          │ │      주간 DAU 추이         │  │
│  │          [차트]                │ │         [차트]             │  │
│  └────────────────────────────────┘ └────────────────────────────┘  │
│                                                                     │
│  ┌────────────────────────────────┐ ┌────────────────────────────┐  │
│  │       최근 신고 대기 (5)        │ │      오늘 경기 (12)        │  │
│  │  - 게시글 #123 욕설 (3건)      │ │  축구: 8경기               │  │
│  │  - 댓글 #456 스팸 (2건)        │ │  야구: 4경기               │  │
│  │  ...                          │ │  ...                       │  │
│  └────────────────────────────────┘ └────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 2.2 핵심 지표 카드

| 지표 | 설명 | 비교 기준 |
|------|------|-----------|
| 오늘 가입 | 금일 신규 가입자 수 | 전일 대비 |
| 오늘 DAU | 금일 활성 사용자 수 | 전일 대비 |
| 오늘 게시글 | 금일 등록 게시글 수 | 전일 대비 |
| 오늘 매출 | 금일 유료 픽 판매액 | 전일 대비 |

### 2.3 빠른 링크 위젯

| 위젯 | 내용 | 액션 |
|------|------|------|
| 신고 대기 | 미처리 신고 건수 및 목록 | 클릭 시 신고 관리 이동 |
| 오늘 경기 | 오늘 예정/진행 경기 요약 | 클릭 시 경기 관리 이동 |
| 푸시 대기 | 예약된 푸시 발송 건수 | 클릭 시 푸시 관리 이동 |
| 최근 문의 | 1:1 문의 미답변 건수 | 클릭 시 문의 관리 이동 |

### 3.2 유저 통계 지표

| 지표 | 설명 | 기간 |
|------|------|------|
| 총 회원 수 | 전체 가입 회원 (탈퇴 제외) | 누적 |
| 신규 가입 | 기간 내 신규 가입자 | 일/주/월 |
| 탈퇴 | 기간 내 탈퇴 회원 | 일/주/월 |
| 정지 회원 | 현재 정지 상태 회원 | 현재 |
| DAU | Daily Active Users | 일별 |
| WAU | Weekly Active Users | 주별 |
| MAU | Monthly Active Users | 월별 |
| 리텐션 | 재방문율 | D1, D7, D30 |

### 3.3 유저 세그먼트 분석

| 세그먼트 | 기준 |
|----------|------|
| 신규 유저 | 가입 7일 이내 |
| 활성 유저 | 최근 7일 내 접속 |
| 휴면 유저 | 30일 이상 미접속 |
| 이탈 위험 | 14일 이상 미접속 |
| VIP | 유료 픽 구매 이력 |
